# scrapy
This is a example web scraping

## Requirements
* Scrapy 2.13.3+ 
* Python 3.13.1+

## Instruction

```bash
git clone https://github.com/MIbarra360/scrapy.git
pip install -r requirements.txt
scrapy crawl webscrap
```
